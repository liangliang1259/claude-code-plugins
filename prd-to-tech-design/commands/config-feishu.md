# 配置飞书API凭据

此命令帮助设置飞书API凭据，以便直接从飞书获取文档。

## 步骤

1. **检查现有配置**
   - 检查 `config/feishu_config.json` 是否存在
   - 显示当前配置状态

2. **引导用户完成设置**
   - 询问飞书 app_id
   - 询问飞书 app_secret
   - 可选：询问自定义API基础URL（默认：https://open.feishu.cn）

3. **验证凭据**
   - 使用飞书客户端测试凭据
   - 验证对文档API的访问权限

4. **保存配置**
   - 创建 `config/feishu_config.json`
   - 设置适当的文件权限（600）
   - 如可能，进行加密存储

5. **提供后续步骤**
   - 说明如何使用文档ID获取功能
   - 提供飞书文档URL示例

## 错误处理

- 无效凭据：提供清晰的错误消息和设置说明
- 权限不足：列出所需的飞书应用权限
- 网络问题：建议故障排除步骤

## 所需飞书应用权限

确保您的飞书应用具有以下权限：
- `docx:document` - 查看、评论、导出和管理文档
- `docx:document:readonly` - 只读访问文档

## 获取凭据步骤

1. 访问 https://open.feishu.cn/app
2. 创建自建应用或使用现有应用
3. 在应用详情页获取 App ID 和 App Secret
4. 在权限管理中添加文档权限
5. 发布应用版本
